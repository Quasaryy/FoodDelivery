# Tech Stack
- **Language:** SWIFT 5
- **IDE:** Xcode 15
- **iOS:** 17
- **Archetecture:** MVVM
- **Frameworks:**  Core Location, Core Animation, UIKit, Foundation
- **Layout:** From Figma, Interface Builder, Xib
- **GCD:** Yes 
- **Features:** 
  - Custom Navigation bar
  - Shoppping cart
  - Custom Tab Bar
  - Working with user location
  - Custom animation
  - Custom Stepper
  - Custom modal window
  - Custom fonts
  - URLSession
  - DataModel
  - Rest API
  - Data models
  - Json decoding and ecoding
  - TableView
  - CollectionView
  - NotificationCenter
  - Protocols
  - Delegates
  - And many more
